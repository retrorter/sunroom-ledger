# UNIFIED TOKEN GLOSSARY & SPATIAL TAXONOMY (v2.5.2)

## 1. PRIMARY LOCATION CODES
- hb  :: House Bed (Outdoor front container crops / steps)
- bb  :: Basement Bed (Outdoor lower terrace growth zone)
- sr  :: Sunroom (Main indoor microclimate array)
- lr  :: Living Room (New indoor expansion tracking tier)
- la  :: Light Arrays (Decoupled, mobile hardware footprint tracking)

## 2. WINDOW & ORIENTATION DESCRIPTORS
- nnw :: North-North-West Vector (The Long Wall tracking plane)
- ene :: East-North-East Vector (The Front Wall solar engine)
- sse :: South-South-East Vector (The Right Wall doorway plane)

## 3. CAMERA FRAMING METADATA
- cu  :: Close-Up (Macro detail, single petioles, hydathodes, pests)
- mcu :: Medium Close-Up (Individual plant focus, complete node views)
- ms  :: Medium Shot (Zonal views, shelf arrays, immediate context)
- ws  :: Wide Shot (Global room perspective, broad light vectors)

## 4. STANDARD FILE FORMATTING SPECIFICATION
- Data Logs:  YYYY-MM-DD_<location>-<zone>_par.md
- Media Assets: <location-node>-<context-descriptor>-<framing>-<timestamp-suffix>.jpg
